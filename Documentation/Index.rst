Video Url replace
=============================================================

.. only:: html

	:Classification:
		brt_videourlreplace

	:Version:
		|release|

	:Language:
		en

	:Description:
		replaces video URLs from Youtube, Vimeo and Dailymotion with a videothumbnail, which is replaced by a responsive iframe when clicked. 
		uses the Bootstrap responsive embed class.
		Inspired by: http://www.sitepoint.com/faster-youtube-embeds-javascript/
		
		Requires (not included):
			- jQuery
			- Bootstrap
